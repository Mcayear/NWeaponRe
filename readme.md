## 如何编译

安装TypeScript
```
npm install -g typescript
```

从远程仓库中拉取
```
git clone https://github.com/Mcayear/NWeaponRe.git
```

前往目录
```
cd NWeaponRe
```

执行编译
```
npm run comp
```

### 快速构建
你需要先安装打包所需的第三方archiver包，可以直接
```
npm install
```

然后执行构建
```
npm run build
```

构建完成后可以在`./dist/@NWeaponRe.zip`找到压缩包
